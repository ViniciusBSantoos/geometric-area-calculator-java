# 📐 Sistema de Cálculo de Área de Figuras Geométricas em Java

Projeto desenvolvido para calcular a área de diferentes figuras geométricas utilizando conceitos modernos de Programação Orientada a Objetos (POO), como **interfaces** e **expressões lambda**.

---

## 🚀 Funcionalidades

* Cálculo de área das seguintes figuras:

  * Quadrado
  * Retângulo
  * Círculo

* Exibição do resultado formatado (2 casas decimais)

* Associação do nome da figura com seu cálculo utilizando `Map`

---

## 📋 Regras de Negócio

Cada figura possui sua própria fórmula de cálculo:

* 🟦 Quadrado → Área = lado × lado
* ▭ Retângulo → Área = base × altura
* ⚪ Círculo → Área = π × raio²

---

## 🧠 Conceitos aplicados

* Interface (`interface`)
* Interface funcional
* Expressões Lambda (`->`)
* Polimorfismo
* Estrutura de dados (`Map`)
* Formatação de números (`printf`)

---

## 🛠 Tecnologias Utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Java 8+ (Lambda)
